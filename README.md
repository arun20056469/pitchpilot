# PitchPilot

PitchPilot is a React and TypeScript project that helps developer candidates prepare a clear 5-7 minute project explanation video. It turns the video preparation process into a polished workspace with a protected setup screen, reusable dashboard components, technical contribution checklists, challenge stories, rehearsal timing, local persistence, and an auto-generated speaking guide.

## Features

- Candidate setup flow with role and project selection
- Protected dashboard experience after setup
- Storyline builder for introduction, overview, contribution, implementation, challenges, learnings, and closing
- Technical checklist for frontend and backend explanation points
- Rehearsal timer aligned to the 5-7 minute target
- Editable challenge cards using the problem, cause, solution, and learning format
- Auto-generated speaking guide for the video submission
- Local storage persistence through an async mock API layer
- Responsive UI for mobile and desktop screens

## Tech Stack

- React
- TypeScript
- Vite
- Context API and reducer state management
- Local storage persistence
- Lucide React icons
- CSS responsive layout

## Run Locally

npm run dev
```

```

## Video Explanation Points

Use `VIDEO_EXPLANATION_GUIDE.md` for a quick 5-7 minute explanation flow. The most important points to mention are:

- You built a frontend project called PitchPilot for candidates preparing project explanation videos.
- You implemented reusable React components for dashboard cards, story segments, checklists, timer, and guide generation.
- You used Context API with a typed reducer to manage app state consistently.
- You simulated API behavior through async service functions and persisted data in local storage.
- You implemented a protected workspace flow where the dashboard appears only after candidate setup.
- You improved UX with responsive layouts, clear timing feedback, and editable preparation notes.
